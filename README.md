# I2c::AlphaDisplay

## Installation

    $ gem install i2c-alpha_display

## Usage

Pipe any output to `i2cdisp` and it will do its best to show it on the display. Numberic input will have siffixes added so that it fits on the display. Eg. `12345` => `12.3k`

    $ echo helo | i2cdisp
